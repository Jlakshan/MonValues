What is MonValues?

MonValues is a political test specifically made for Monarchists of all sorts.

What are the 28 values?

There are 14 independent axes, and each has two opposing values assigned to them :

Constitutionalism ( Powers )

Absolutism ( Powers )

Democracy ( Democracy )

Autocracy ( Democracy )

Liberty ( Liberty )

Security ( Liberty )

Decentralisation ( Centralisation )

Centralisation ( Centralisation )

Localism ( Localism )

Unionism ( Localism )

Isolationism ( Openness )

Globalism ( Openness )

Nationalism ( Nationalism )

Internationalism ( Nationalism )

Militarism ( Conflict )

Pacifism ( Conflict )

Traditionalism ( Values )

Progressivism ( Values )

Monoculturalism ( Culture )

Multiculturalism ( Culture )

Theocracy ( Religion )

Laicism ( Religion )

Etatism ( Ownership and Trade )

Laissez-faire ( Ownership and Trade )

Regulationism ( Regulation )

Deregulationism ( Regulation )

Production ( Production )

Sustainability ( Production )
